## 關鍵重點
- 先判斷「這是不是正確學習解法」，而不是先挑老師或課程；CDC 把 needs assessment 放在訓練品質第一項。（來源 1：https://www.cdc.gov/training-development/php/qts/index.html）
- 線上課程可看：能力對應、學習目標、持續教育或證書、發布/審查日期、學員回饋、是否經品質審查。（來源 2：https://www.cdc.gov/training-development/php/qts/indicating-training-quality-to-learners.html）
- MOOC / 線上課程的學習者回饋研究中，課程影片、練習、教學排程、教材呈現是關鍵品質因素。（來源 3：https://www.sciencedirect.com/org/science/article/pii/S1539310025000079）
- 選教練時，ICF 的可驗證維度包括 credential、training、experience、ethics，以及 specialty / language / location 等匹配條件。（來源 4：https://icfsupport.freshdesk.com/support/solutions/articles/5000881265-what-is-the-credentialed-coach-finder-ccf-）
- 選 mentor / 老師型人物時，Yale 的框架偏向：是否敬佩其成就與經驗、是否能合作溝通、是否能導向你的專業目標、對方是否真的喜歡自己的職涯。（來源 5：https://your.yale.edu/working-at-yale/learn-and-grow/career-development/mentoring/how-to-choose-a-mentor）
- 選講者時，近期產業資料強調先定義活動目標與受眾需求，再判斷講者是否能對應 inspiration、strategy、skill building 等不同目的。（來源 6：https://speakers.com/book-a-keynote-speaker/）

## 不確定點
- 無資料：我沒有找到一篇近期、權威、直接統合「老師、線上課程、講者、教練」四類對象的單一決策框架。
- coaching credential 的解讀有爭議：ICF credential 是可驗證品質訊號，但不等於所有有效教練都必須持有 ICF credential。
- 講者選擇的近期資料多來自 speaker bureau 或商業網站，代表實務觀點，但不是中立學術標準。
